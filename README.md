# TourismMap

Цей проєкт створено для пошуку маршрутів для спортивних гірських походів. На основі бази даних долин та перевалів Кавказу будується граф сполучень між перевалами та долинами, а за допомогою алгоритма пошуку A* будуються декілька варіантів найбільш оптимальних для проходження маршрутів відповідної спортивної складності. 3D-візуалізацію реалізовано за допомогою MapBox API.
